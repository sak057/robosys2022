# plusコマンド
![test](https://github.com/sak057/robosys2022/actions/workflows/test.yml/badge.svg)



標準入力から読み込んだ数字を足す。
使い方  
①　コードを使用する端末に入力する。（ここではディレクトリ名をplusとする。）  
②　計算したい数字をseqもしくはファイルに記述する。  
③　②で記述したものを指定して./plusを実行し、結果を出す。  

## 必要なソフトウェア
* Python
* テスト済み: 3.7～3.10

## テスト環境
* Ubuntu22.04LTS
